
Локатор — обычный текст, которой идентифицирует себя как элемент DOM’а страницы. 
Простым языком: с помощью локатора на странице можно найти элементы.
Есть очень много видов локаторов, но чаще всего в работе применяется лишь часть из них. 
Их можно искать по следующим видам:
-> имя элемента
-> id
-> классы
-> кастомные атрибуты
-> родители и дети элементов
-> ссылки
-> другие.


-------------- Примеры: ---------------
.form-control - элемент с классом form-control

----------- ТИПЫ CSS-селекторов -----------
1) По классу: .class_name
.form-control - элемент с классом form-control

2) По идентификатору: #id_name
#test - элемент с id="test"

3) По атрибуту:
3.1. [атрибут="A"]: атрибут состоит из A
input[type="text"] - найти элемент input с атрибутом type, у которого значение = "text"
[type="text"] - найти ВСЕ элементы (любые) с атрибутом type, у которого значение = "text"

3.2. [атрибут$="A"]: атрибут оканчивается из A
img[src$=".png"] - найти все теги img, которые имеют src, который оканчивается на ".png"

3.3. [атрибут*="A"]: атрибут содержит A (не важно, где эта подстрока находится - в начале, 
в середине или в конце).
img[src*=".p"] - найти все теги img, которые имеют src, который содержит подстроку ".p"

3.4. [атрибут^="A"]: атрибут начинается с A
input[type^="t"] - найти все элементы input с атрибутом type, значение которого начинается на "t".
img[src^="photo"] - найти все теги img с атрибутом src, значение которого начинается на "photo"
